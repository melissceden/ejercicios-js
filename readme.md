# Ejercicios de JavaScript

Este repositorio existe para guardar mis prácticas y soluciones de ejercicios en JavaScript.  
Aquí iré subiendo cada archivo a medida que avance, sin un orden específico, simplemente como registro de aprendizaje.

Los ejercicios incluyen temas como lógica, funciones, arreglos, ciclos, condicionales y otros conceptos básicos e intermedios. La mayoría muestran sus resultados en la consola con `console.log()`.

El propósito es mantener un historial claro de mi progreso y poder revisar, mejorar y comparar mis soluciones con el tiempo. El repositorio se actualizará constantemente conforme vaya resolviendo nuevos ejercicios.
